# Level

An integer log level to write to the PMS log with.
  - 0: Error
  - 1: Warning
  - 2: Info
  - 3: Debug
  - 4: Verbose

## Example Usage

```java
import dev.plexapi.sdk.models.operations.Level;

Level value = Level.ZERO;
```
