# CollectionMode

Display mode for the collection.

## Example Usage

```java
import dev.plexapi.sdk.models.shared.CollectionMode;

CollectionMode value = CollectionMode.DEFAULT;
```


## Values

| Name         | Value        |
| ------------ | ------------ |
| `DEFAULT`    | default      |
| `HIDE_ITEMS` | hideItems    |
| `SHOW_ITEMS` | showItems    |
